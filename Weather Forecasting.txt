Weather forevasting to gra, w ktorej mozesz przewidziec/zgadywac jaka bedzie pogoda za x czasu. 

Użytkownik może sprawdzic jaka pogoda (temperatura, kierunek wiatru, opady) była lub jest w danym mieście. Ma też dostęp do mapy.
 
ZASADY:
Musi on ustalić na podstawie dostępnych danych jaka pogoda będzie za określony przez niego czas w danym mieście.

PUNKTY:
Im lepiej przewidzi pogode i im późniejszą datę wybierze to ma więcej punktów.

- dodawanie predykcji pogodowej,

- ogólny score board,

DTO - Data Transfer Object - to obiekt, za pośrednictwem, którego udostępnia się jakieś dane zwykle na zewnątrz aplikacji - suche dane. 
DAO - Data Access Object - czyli coś co zapewnia dostęp do źródła danych, nie ważne czy to z pliku czy to z bazy danych.

interface PersonDTO {
    String getName();
    void setName(String name);
    //.....
}

interface PersonDAO {
    PersonDTO findById(long id);
    void save(PersonDTO person);
    //.....
}
